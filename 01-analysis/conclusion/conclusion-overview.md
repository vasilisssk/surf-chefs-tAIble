# Заключение по проекту "Кулинарная студия"

## Обзор проекта

Проект "Кулинарная студия" представляет собой мобильное Android-приложение, предназначенное для организации и управления записью на кулинарные мастер-классы. Приложение предоставляет пользователям возможность просматривать расписание классов, записываться на них, управлять бронированиями и взаимодействовать с поварами.

## Архитектура приложения

### Общая архитектура
- Основное приложение реализуется с использованием Bottom Navigation Activity
- Аутентификация (регистрация/вход) реализуется на отдельных Activity
- После успешной аутентификации пользователь направляется на основную Activity с Bottom Navigation

### Навигационная структура
1. **Аутентификационные экраны** (отдельные Activity):
   - Экран регистрации
   - Экран входа

2. **Основная навигация** (Bottom Navigation Activity):
   - Главная (Home)
   - Расписание (Schedule)
   - Мои бронирования (My Bookings)
   - Профиль (Profile)

## Интеграции с API

Приложение интегрируется с REST API по следующим основным маршрутам:
- `/auth` - аутентификация пользователей
- `/profile` - управление профилем пользователя
- `/classes` - получение информации о классах
- `/bookings` - управление бронированиями
- `/chefs` - информация о поварах
- `/reviews` - система отзывов
- `/rental-packages` - аренда оборудования

## Потоки взаимодействия

### Поток бронирования:
1. Пользователь на Schedule Fragment видит класс
2. Нажимает "Забронировать" на классе
3. Открывается Class Detail Fragment (bottom sheet)
4. Нажимает "Подтвердить бронирование"
5. Открывается Booking Fragment (bottom sheet)
6. Подтверждает бронирование
7. Бронирование создается и появляется в My Bookings Fragment

### Поток оценки шефа:
1. После завершения класса (по времени или по уведомлению)
2. Пользователь заходит в My Bookings Fragment
3. Видит завершенное бронирование
4. Нажимает на него -> Booking Detail
5. Видит кнопку "Оценить шефа"
6. Нажимает - открывается форма оценки (внутри Booking Detail как bottom sheet)
7. Выставляет оценку (1-5 звезд)

Архитектура приложения обеспечивает удобство использования и гибкость в дальнейшем развитии.
