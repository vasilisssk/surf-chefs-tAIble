## Обзор

Бизнес-логика бронирования управляет процессами создания, получения, обновления и отмены бронирований в приложении "Кулинарная студия". Логика обеспечивает проверку доступности мест, обработку ошибок и синхронизацию состояния бронирований с сервером.

### User Story

> Как пользователь, я хочу создавать и управлять своими бронированиями,
> чтобы планировать посещение кулинарных классов.

### Бизнес-ценность

- Упрощение процесса бронирования для пользователей
- Управление доступными местами в классах
- Обеспечение точности бронирований и предотвращение перепродаж

## Описание логики

### Создание бронирования

Процесс создания бронирования включает проверку доступности мест в выбранном классе, отправку данных на сервер и обновление локального состояния.

### Отмена бронирования

Процесс отмены бронирования включает проверку возможности отмены (за 24 часа до начала), отправку запроса на сервер и обновление состояния.

## API запросы

### POST /bookings

**Триггер:** Подтверждение бронирования на экране бронирования

**Headers:**

| Поле | Описание |
|------|----------|
| `authorization` | Bearer токен пользователя |
| `content-type` | application/json |

**Параметры/Body:**

| Параметр | Тип | Описание | Значение/Источник |
|----------|-----|----------|-------------------|
| `classId` | string | ID кулинарного класса | `{classId}` |
| `rentalPackageId` | string | ID пакета проката | `{rentalPackageId}` |
| `allergies` | string | Информация об аллергиях | `{allergies}` или null |

**Обработка ответа:**

| Результат | Действие |
|-----------|----------|
| Загрузка | Индикатор загрузки на кнопке, блокировка UI |
| Успех (201) | Обновление списка бронирований, переход к экрану "Мои бронирования" |
| Ошибка 400 | Отображение ошибки "Невалидные данные или нет доступных мест" |
| Ошибка 401 | Переход к экрану входа |
| Ошибка 409 | Отображение ошибки "Бронирование конфликтует с существующим" |
| Ошибка 5xx | Снек "Произошла ошибка. Попробуйте позже" |
| Ошибка сети | Снек "Нет соединения. Проверьте подключение к интернету" |

---

### PUT /bookings/{bookingId}

**Триггер:** Отмена бронирования в списке бронирований

**Headers:**

| Поле | Описание |
|------|----------|
| `authorization` | Bearer токен пользователя |
| `content-type` | application/json |

**Параметры/Body:**

| Параметр | Тип | Описание | Значение/Источник |
|----------|-----|----------|-------------------|
| `action` | string | Действие с бронированием | "cancel" |
| `reason` | string | Причина отмены | Ввод пользователя или null |

**Обработка ответа:**

| Результат | Действие |
|-----------|----------|
| Загрузка | Индикатор загрузки на кнопке |
| Успех (200) | Обновление состояния бронирования в списке |
| Ошибка 400 | Отображение ошибки "Невозможно отменить бронирование" (слишком поздно) |
| Ошибка 401 | Переход к экрану входа |
| Ошибка 404 | Отображение ошибки "Бронирование не найдено" |
| Ошибка 5xx | Снек "Произошла ошибка. Попробуйте позже" |
| Ошибка сети | Снек "Нет соединения. Проверьте подключение к интернету" |

## Критерии приёмки

| ID | Критерий |
|----|----------|
| AC-001 | **Дано** пользователь на экране бронирования с доступным классом, **Когда** подтверждает бронирование, **Тогда** создается новая запись в системе |
| AC-002 | **Дано** пользователь имеет активное бронирование, **Когда** отменяет его за 24+ часа до начала, **Тогда** бронирование отменяется успешно |
| AC-003 | **Дано** пользователь заходит в "Мои бронирования", **Когда** открывает экран, **Тогда** видит актуальный список своих бронирований |
| AC-N01 | **Дано** пользователь пытается забронировать класс без мест, **Когда** подтверждает бронирование, **Тогда** отображается сообщение об ошибке |
| AC-N02 | **Дано** пользователь пытается отменить бронирование менее чем за 24 часа, **Когда** нажимает отмену, **Тогда** отображается сообщение о невозможности отмены |

## Обработка ошибок

| Тип ошибки | Контекст | Действие |
|------------|----------|----------|
| Нет доступных мест | Создание бронирования | Отображение сообщения "Нет доступных мест в выбранном классе" |
| Невозможно отменить | Отмена бронирования | Отображение сообщения "Невозможно отменить бронирование менее чем за 24 часа до начала" |
| Ошибка сети | Любой запрос | Отображение сообщения "Нет соединения. Проверьте подключение к интернету" |
| Серверная ошибка | Любой запрос | Отображение сообщения "Произошла ошибка. Попробуйте позже" |
| Неавторизованный доступ | Любой запрос | Переход на экран входа |
