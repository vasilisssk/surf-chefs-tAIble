## Обзор

Бизнес-логика отзывов управляет процессами создания и отображения отзывов о шефах в приложении "Кулинарная студия". Логика обеспечивает проверку возможности оставления отзыва (только после посещения класса), отправку данных на сервер и обновление рейтинга шефов.

### User Story

> Как пользователь, я хочу оставлять отзывы о шефах,
> чтобы делиться своим опытом и помогать другим пользователям.

### Бизнес-ценность

- Сбор обратной связи для улучшения качества услуг
- Повышение доверия пользователей к шефам
- Создание системы рейтинга для выбора классов

## Описание логики

### Создание отзыва

Процесс создания отзыва включает проверку возможности (только для завершенных бронирований), сбор данных отзыва и отправку на сервер.

---

## API запросы

### POST /reviews

**Триггер:** Отправка формы отзыва

**Headers:**

| Поле | Описание |
|------|----------|
| `authorization` | Bearer токен пользователя |
| `content-type` | application/json |

**Параметры/Body:**

| Параметр | Тип | Описание | Значение/Источник |
|----------|-----|----------|-------------------|
| `bookingId` | string | ID бронирования | `{bookingId}` |
| `chefId` | string | ID шефа | `{chefId}` |
| `rating` | integer | Оценка от 1 до 5 | `{rating}` |
| `comment` | string | Комментарий к отзыву | `{comment}` или null |

**Обработка ответа:**

| Результат | Действие |
|-----------|----------|
| Загрузка | Индикатор загрузки на кнопке, блокировка UI |
| Успех (201) | Отображение сообщения об успехе, обновление рейтинга шефа |
| Ошибка 400 | Отображение ошибки валидации данных |
| Ошибка 401 | Переход к экрану входа |
| Ошибка 409 | Отображение ошибки "Отзыв уже существует для этого бронирования" |
| Ошибка 5xx | Снек "Произошла ошибка. Попробуйте позже" |
| Ошибка сети | Снек "Нет соединения. Проверьте подключение к интернету" |

## Критерии приёмки

| ID | Критерий |
|----|----------|
| AC-001 | **Дано** пользователь завершил бронирование, **Когда** нажимает "Оценить шефа", **Тогда** открывается форма создания отзыва |
| AC-002 | **Дано** пользователь заполняет форму отзыва, **Когда** отправляет её, **Тогда** отзыв создается и обновляется рейтинг шефа |
| AC-003 | **Дано** пользователь пытается оставить отзыв дважды, **Когда** отправляет второй отзыв, **Тогда** отображается ошибка дублирования |
| AC-N01 | **Дано** пользователь пытается оставить отзыв на незавершенное бронирование, **Когда** пытается открыть форму, **Тогда** кнопка "Оценить шефа" недоступна |
| AC-N02 | **Дано** пользователь вводит некорректные данные отзыва, **Когда** отправляет форму, **Тогда** отображается сообщение об ошибке валидации |

## Обработка ошибок

| Тип ошибки | Контекст | Действие |
|------------|----------|----------|
| Отзыв уже существует | Создание отзыва | Отображение сообщения "Отзыв уже оставлен для этого бронирования" |
| Некорректная оценка | Создание отзыва | Отображение сообщения об ошибке валидации |
| Ошибка сети | Любой запрос | Отображение сообщения "Нет соединения. Проверьте подключение к интернету" |
| Серверная ошибка | Любой запрос | Отображение сообщения "Произошла ошибка. Попробуйте позже" |
| Неавторизованный доступ | Любой запрос | Переход на экран входа |
