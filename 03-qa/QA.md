## Использованные инструменты: Android Studio | Qoder (lite mode)

## Промпты:
1. помоги составить промпт для описания тестов на визуальную компоновку экранов. фронт - [frontend](app/src/main), бэк - [backend](02-development/backend). Напиши это в md-файл в [03-qa](03-qa) в отдельную папку prompts
2. можешь теперь создать промпт для создания тестов (компонентные, функциональные, uc-проверки). также в отдельную папку prompts
3. можешь визуальные тесты сделать в виде таблицы в md-файд со следующими колонками: экран | компонент | планируемые layout-проверки | приоритет
4. можешь из этих файлов теперь два json-файлы для каждого тест-кейса. файл должен представлять собой список тест-кейсов со следующимм полями: описание, лейблы (список категорий и тэгов, которые характеризуют тест-кейс), цель, прикрепляемые файлы, приоритет, предполагаемое время выполнения тест-кейса в минутах, путь в котором хранится тест-кейс, шаги и данные
