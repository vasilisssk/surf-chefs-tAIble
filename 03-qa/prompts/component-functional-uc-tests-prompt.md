# Промпт для компонентных, функциональных и UC-тестов

## Контекст проекта

**Приложение:** Chefs tAIble — Android-приложение для бронирования кулинарных классов.
**Архитектура:** MVVM (ViewModel + LiveData + Repository), Koin DI, Retrofit.
**Бэкенд:** Spring Boot, JPA/Hibernate, Flyway, JWT (access + refresh tokens).
**Тестовые фреймворки:** JUnit 5 (backend), JUnit 4 + Mockito + Turbine (frontend unit), Espresso (UI).
